# PCA_spaziale_gee
Questo codice GEE esegue una PCA Spaziale su dati radar Sentinel-1 (2017-2024) di un vigneto. Crea un'immagine mediana temporale (VV, VH, RVI, RFDI) e vi applica la PCA per isolare la variabilità spaziale. Genera mappe delle Componenti Principali, utili in agricoltura di precisione per mappare vigoria e stato del suolo.
